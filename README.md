# Executive Credit Card Performance Dashboard

## Overview
An executive-level **Credit Card Analytics Project** that combines **Power BI dashboards** with a **Python data analysis layer** to deliver actionable insights into credit card performance, customer behavior, and transaction trends.

This project is built for **portfolio presentation**, **executive reporting**, and **data-driven decision making**.

---

## Business Objectives
- Monitor total revenue, interest, fees, and transaction volume  
- Analyze customer spending behavior and demographics  
- Track key KPIs for executive-level reporting  
- Identify growth opportunities and risk areas  
- Support strategic business decisions using analytics  

---

## Dataset
- **Source:** Credit card transactions & customer data (CSV)
- **Tables:**
  - `credit_card_transactions.csv`
  - `credit_card_customers.csv`

> ⚠️ Dataset is **sample/demo data** for portfolio and educational purposes only.

---

## Key KPIs
- Total Revenue  
- Total Interest  
- Total Transaction Amount  
- Annual Fees  
- Total Transactions Count  
- Customer Satisfaction Score
